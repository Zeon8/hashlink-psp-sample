# HashLink PSP Demo
A sample PSP game created using Haxe and HashLink and raylib. 

![Image](https://github.com/user-attachments/assets/082677e7-f3a7-49a6-a9d6-c4c5d5650a3e)

## Building
Make sure you have Haxe and HashLink installed.

### Windows
1. Download [PSPSDK docker image](https://pspdev.github.io/installation/docker.html)
2. Install HashLink for PSP: 
```bash
haxelib git hashlink-psp https://github.com/Zeon8/hashlink psp
```
3. Run `compile_docker.bat` or `compile_podman.bat`

### Linux / MacOS
1. Install PSPSDK
2. Install HashLink for PSP:
```bash
haxelib git hashlink-psp https://github.com/Zeon8/hashlink psp
```
3. Run `compile.sh`
